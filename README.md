## About

DJ software for music management and performance.

## Features

- Track analysis
- Playlists
- Waveforms
- Beat sync
- Effects

## Installation

### From App Store
Open the **App Store** in zOS and search for "Rekordbox".

### Via npm
```bash
npm install @zos-apps/rekordbox
```

## Usage

- Import tracks
- Analyze BPM
- Create DJ playlists



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
